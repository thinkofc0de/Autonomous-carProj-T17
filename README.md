This project presents a Real-Time AI-Based Autonomous Vehicle System designed for complex 
Indian road conditions. It integrates computer vision and machine learning for lane tracking, 
obstacle detection, and road condition analysis, including potholes and damaged surfaces. The 
system enhances safety through crowd-aware navigation by detecting pedestrians, animals, and 
traffic density, while dynamically adjusting speed and braking. It also features emergency vehicle 
detection with automatic yielding using audio and visual cues. Additionally, road hazards are 
logged to an IoT-based cloud platform for smart-city monitoring. Overall, the system combines 
autonomous navigation with real-world adaptability, safety, and intelligent decision-making.

PROPOSED SYSTEM

The proposed system focuses on developing an autonomous vehicle that uses computer vision to
analyze road conditions and assist in intelligent navigation.A camera mounted on the vehicle
captures real-time images of the road environment,which are processed using image processing
techniques and machine learning algorithms.The system detects roadlanes, traffic signs,vehicles,
pedestrians, and other obstacles.Based on this analysis, the vehicle can make driving decisions
such as steering, slowing down, or stopping automatically.This approach improves roadsafety,
reduces humanerrors,and supports the development of smart transportation systems.
